# 🍻 Tavern of Legends

Welcome to **Tavern of Legends**, a premium, addictive 16-bit fantasy character summoning RPG! Test your luck, gather an elite vanguard of legendary heroes, claim bounties, discover ancient artifacts, and conquer deep dungeon floors in this polished incremental adventure.

## 🚀 Play Instantly Online!

No setup, no terminal, and no installs needed! You can play **Tavern of Legends** directly in your web browser:

👉 **[Play Tavern of Legends Instantly Here!](https://VictorVB18.github.io/tavern-legend/)**

---

## 🎮 Game Features

- **Fate summoning pool**: Roll the dice of fate to recruit from over **34 unique heroes** spanning 7 rarity classes (Common, Uncommon, Rare, Epic, Legendary, Mythic, Divine).
- **Beautiful Premium Pixel Art**: Fully transparent 16-bit pixel art styles for all characters.
- **Hero Collection Index**: Track your discoveries and unlock clean dark silhouettes of locked heroes.
- **Guild Quests & Tome of Rules**: Page-turning quest book interface to claim gold bounties and permanent rewards.
- **Dungeon Boss Crawls**: Select up to 3 of your strongest heroes and test their collective power in epic boss battles.
- **Ancient Artifact Equipment**: Recover relics like the *Sword of the Ancients* from defeated bosses and equip them to amplify individual hero stats.
- **Prestige System**: Reset your gold and upgrades to gain a permanent global **2x fate multiplier**!
- **Auto-Summoning & Smart Dismissing**: Automate your summons and toggle low-rarity auto-dismiss gold generation.

## 🛠️ Local Development (For Developers)

If you want to run the game locally on your computer and edit the code, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### 1. Clone the repository
```bash
git clone https://github.com/VictorVB18/tavern-legend.git
cd tavern-legend
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run development server
```bash
npm run dev
```
This will start a local server at `http://localhost:5173/` where you can play and see live code changes!

### 4. Build for production
```bash
npm run build
```

## 💾 How Game Saves Work

Your journey is perfectly safe! The game automatically saves your progress (Gold, Summons, Vanguard, Upgrades, Quests, Relics, and Prestige) directly to your browser's **Local Storage** in real-time.
- This works **both online** (on the GitHub Pages link) and **locally** (on your computer).
- You can close the tab or turn off your computer, and your progress will be waiting for you when you return!
- If you play locally, your progress is tied to your local browser slot.

## 🔄 How to Update Your Local Copy (Fast & Easy)

Since we are actively updating the game with new features and characters, you **do not** need to re-install everything when there is a new update! 

Once you have done the initial installation, updating and playing takes less than 5 seconds:

### To get new updates and play:
1. Open your terminal in the project folder.
2. Pull the latest updates:
   ```bash
   git pull
   ```
3. Start the game instantly:
   ```bash
   npm run dev
   ```
*(No need to run `npm install` again unless we introduce new external packages!)*
